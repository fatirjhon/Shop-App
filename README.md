## Requirements

- [Node.js](https://nodejs.org/en/) 6+

```shell
npm install
```


## Running

Make sure to add a `config.js` file in the `config` folder. See the example there for more details.

Set MongodDB to your default path:

```shell
mongod --dbpath 'your mongodata path'
```

Development mode:

```shell
npm run start:dev
```
